Returns the number of the most recently mined block.

## Parameters

No parameters.

## Returns

<ResponseField name="result" type="string">
  The current block number as a hexadecimal string.
</ResponseField>

## Example

<CodeGroup>
```json Request
{
  "jsonrpc": "2.0",
  "method": "eth_blockNumber",
  "params": [],
  "id": 1
}
```

```json Response
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x158a0e9"
}
```
</CodeGroup>

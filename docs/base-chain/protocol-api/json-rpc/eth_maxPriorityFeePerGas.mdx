---
title: 'eth_maxPriorityFeePerGas'
sidebarTitle: 'eth_maxPriorityFeePerGas'
description: 'Returns the suggested EIP-1559 priority fee (tip) per gas.'
---

Returns a suggested value for `maxPriorityFeePerGas` to use in an EIP-1559 transaction. This is the tip paid to the sequencer on top of the base fee.

## Parameters

No parameters.

## Returns

<ResponseField name="result" type="string">
  The suggested priority fee per gas in wei as a hexadecimal string.
</ResponseField>

## Example

<CodeGroup>
```json Request
{
  "jsonrpc": "2.0",
  "method": "eth_maxPriorityFeePerGas",
  "params": [],
  "id": 1
}
```

```json Response
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x1"
}
```
</CodeGroup>

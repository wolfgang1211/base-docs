---
title: 'web3_clientVersion'
sidebarTitle: 'web3_clientVersion'
description: 'Returns the current client version string.'
---

Returns the version string of the node client software.

## Parameters

No parameters.

## Returns

<ResponseField name="result" type="string">
  The client version string.
</ResponseField>

## Example

<CodeGroup>
```json Request
{
  "jsonrpc": "2.0",
  "method": "web3_clientVersion",
  "params": [],
  "id": 1
}
```

```json Response
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "op-reth/v1.1.0/linux-x86_64"
}
```
</CodeGroup>

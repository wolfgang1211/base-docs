Returns the chain ID of the current network per [EIP-695](https://eips.ethereum.org/EIPS/eip-695).

## Parameters

No parameters.

## Returns

<ResponseField name="result" type="string">
  The chain ID as a hexadecimal string. `"0x2105"` (8453) for Base Mainnet, `"0x14a34"` (84532) for Base Sepolia.
</ResponseField>

## Example

<CodeGroup>
```json Request
{
  "jsonrpc": "2.0",
  "method": "eth_chainId",
  "params": [],
  "id": 1
}
```

```json Response (Base Mainnet)
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x2105"
}
```

```json Response (Base Sepolia)
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x14a34"
}
```
</CodeGroup>

---
title: "Base Execution Differences"
description: Base-specific execution behavior to account for while building EVM-compatible applications.
---

Base is built to be as compatible as possible with Ethereum, and there are very few differences when it comes to building on Base versus Ethereum.

However, there are still some minor discrepancies between the behavior of Base and Ethereum that you should be aware of when building apps on top of Base.

## Differences

These minor differences include:

- [Flashblocks ordering](/base-chain/network-information/transaction-ordering#flashblocks-ordering)
- [Minimum base fee](/base-chain/network-information/network-fees#minimum-base-fee)
- [Bridging and withdrawals](/base-chain/network-information/bridging-and-withdrawals)
- [Opcodes](https://docs.optimism.io/stack/differences#opcodes)
- [Address aliasing](https://docs.optimism.io/stack/differences#address-aliasing)
- [Transaction costs](https://docs.optimism.io/stack/differences#transaction-fees)
- [Transaction finality](/base-chain/network-information/transaction-finality)
